## EC-Council Exam Simulator

A fully offline, browser-based practice tool for EC-Council certifications.

Built to help security professionals and students practice for exams like **ECIHv3**, **CEH**, and others using their own dump files — without any data leaving the browser.

### Tech Stack

| Component | Technology |
|---|---|
| Site framework | Jekyll + Minima dark theme |
| Hosting | GitHub Pages |
| Simulator | Vanilla JS — no frameworks |
| Excel export | SheetJS (client-side) |
| Data storage | Browser RAM only |

---

### Privacy

This tool processes **everything locally in your browser**. No questions, answers, or results are ever sent to any server. Closing the tab clears all data instantly.
