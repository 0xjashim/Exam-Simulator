# Exam Simulator

> Practice tool for certifications — built on Jekyll Minima dark theme.

**🔗 Live:** https://0xjashim.github.io/Exam-Simulator

---

## Stack
- Jekyll + Minima dark skin
- GitHub Pages (auto-deploy via Actions)
- Vanilla JS simulator — 100% offline
- SheetJS for client-side Excel export

## Local Dev
```bash
bundle install
bundle exec jekyll serve
# → http://localhost:4000/Exam-Simulator
```

## Deploy
Push to `main` → GitHub Actions builds and deploys automatically.
